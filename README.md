# TripCraft – AI Personalized Travel Planner

TripCraft is an AI-powered travel planning application that generates personalized travel itineraries based on user preferences, travel dates, and budget.

The app helps travelers quickly create customized travel plans, discover interesting places, and visualize their routes on an interactive map.

## Live Application

You can try the app here:

TripCraft

## Features

- AI-generated travel itineraries tailored to the user’s preferences

- Personalized recommendations based on travel interests

- Budget-based planning to adapt the trip to different spending levels

- Interactive map showing the travel route and locations

- Tourist attractions with images for each recommended destination

## How It Works

Users provide travel information such as:

- Travel dates

- Budget level

- Travel preferences

The system analyzes the inputs and generates a personalized travel itinerary.

The itinerary includes:

- Recommended places to visit

- A route displayed on a map

- Images for each point of interest.

## Development
